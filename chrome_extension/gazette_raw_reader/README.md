# Gazette Ham Metin Okuyucu

Chrome/Edge kurulumu:

1. `chrome://extensions` sayfasını aç.
2. Sağ üstten `Developer mode` / `Geliştirici modu` aç.
3. `Load unpacked` / `Paketlenmemiş öğe yükle` seç.
4. Bu klasörü seç:
   `chrome_extension/gazette_raw_reader`

Random Haber uygulamasında `Haberi Aç` düğmesine basınca URL `gazette_raw=1` parametresiyle açılır. Extension yüklüyse sayfadaki ana haber metnini sağ tarafta ham metin paneli olarak gösterir.
